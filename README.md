# NovaAnime - Watch HD Anime Online

A modern, responsive anime streaming platform built with HTML, CSS, and vanilla JavaScript.

## Features
- 🎬 Dynamic anime pages from MAL ID
- 📺 Sub & Dub streaming via megaplay.buzz
- 🔍 Live search with filters
- 📋 Continue watching & watchlist
- 🌙 Dark purple neon aesthetic
- 📱 Fully responsive design
- ⚡ No backend required

## Quick Start
1. Clone this repository
2. Deploy to GitHub Pages or Cloudflare Pages
3. Add anime to `data/anime.json` using only `mal_id`
