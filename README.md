# Anime Streaming Website

Modern Anime Streaming Website using:

- HTML
- CSS
- Vanilla JavaScript
- Jikan API
- Megaplay Streaming

## Features

- Dynamic anime pages
- MAL ID system
- Auto episode generation
- Streaming player
- Continue watching
- Watchlist system
- PWA support
- Responsive design
- GitHub Pages support
- Cloudflare Pages support
